# cppp-platfrom -- Portable platform check library

## Build

### Dependencies

+ CMake (version >= 3.10)

### Command

```shell
mkdir build
cd build
cmake .. -DCMAKE_INSTALL_PREFIX=[[PREFIX]]
# If you want to build the example, add -DBUILD_EXAMPLE=ON
cmake --build .
cmake --install .
```
